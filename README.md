# Personalized Webpage

An interactive web application that allows users to enter their personal information, receive customized feedback, and enjoy motivational content. The application leverages JavaScript's core concepts such as DOM manipulation, localStorage, functions, conditional statements, and loops to create a dynamic and personalized user experience.

---

##  Project Overview

The Personalized Webpage is designed to collect a user's name and age through a simple form interface. Once submitted, the information is stored in the browser using localStorage, allowing the data to persist even after refreshing the page.

The application then generates a personalized greeting, calculates the user's age in months, determines whether the user is eligible for adult content based on their age, and displays a motivational quote multiple times using a loop.

---

##  Features

* User-friendly form for collecting personal information
* Persistent data storage using localStorage
* Personalized greeting using template literals
* Age conversion from years to months
* Age verification using if...else statements
* Motivational quotes generated using a for loop
* Dynamic content updates through DOM manipulation

##  How It Works

### User Information Collection

Users enter their name and age through the provided form.

### Data Persistence

The submitted information is stored in localStorage, ensuring the data remains available even after the page is refreshed.

### Personalized Greeting

The application retrieves the stored name and displays a custom greeting message.

### Age Calculation

A JavaScript function converts the user's age from years into months and displays the result.

### Age Verification

An if...else statement checks whether the user is 18 years or older and displays an appropriate message.

### Motivational Quotes

A for loop dynamically generates and displays a motivational quote five times.
